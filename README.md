# PulmoSense AI

PulmoSense AI is a simple Streamlit-based lung cancer risk prediction app. It takes user health information such as age, smoking status, breathing difficulty, SpO₂ level, pollution exposure, family history, energy level, and immune health, then predicts the possible lung cancer risk level using a trained machine learning model.

## Features

- Lung cancer risk prediction
- Risk probability and risk level display
- Interactive Plotly gauge chart
- Detailed factor analysis
- Clinical recommendation
- Downloadable PDF report
- Clean and research-ready user interface

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- XGBoost
- NumPy
- Plotly
- ReportLab
- Joblib
